# Dominos WoTLK

## Install:

1. [Download the package.](https://github.com/bkader/Dominos/archive/refs/heads/main.zip)
2. Open the archive, then extract `Dominos` & `Dominos_Config` folders to `Interface\AddOns`.
3. Enjoy!
